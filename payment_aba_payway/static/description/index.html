ABA PayWay QR Payment for e-Commerce

This module integrates ABA PayWay as a payment provider on your Odoo e-commerce website, allowing customers to pay using QR code at checkout.

Features

  • ABA PayWay QR payment option on the e-commerce checkout page
  • Seamless integration with Odoo's native payment provider framework
  • Real-time payment status updates via ABA PayWay API
  • Supports sandbox mode for testing before going live

Getting Started

  1. Go to Accounting → Configuration → Payment Providers and activate ABA PayWay.
  2. Enter your API Key and Merchant ID as provided by ABA PayWay.
  3. Set the provider to Production mode when ready to go live.

Requirements

  • Python package: cryptography (installed automatically)
