How does it work?

How does it work?

Customer places an order

A customer adds items to their cart and proceeds to checkout on your Odoo eCommerce website.

Selects a payment method

Your customer pays securely with ABA KHQR, Card, WeChat Pay, or Alipay on your Odoo eCommerce website.

Order confirmed

After successful payment, funds settle straight into your ABA account and you can begin fulfilling the order.

How to install?

How to install?

Install the plugin

Install the ABA PayWay plugin from the Odoo App Store on Odoo 18.

Get Sandbox API key

Create a free PayWay Sandbox account to get your test API key.

Configure & Test

Paste your API key into the plugin in Odoo config, then run a test transaction.

Apply to go live

Contact our sales team to set up your ABA PayWay account and get paid.
