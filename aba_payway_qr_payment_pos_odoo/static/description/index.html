ABA PayWay QR Payment for POS

Let customers pay using QR codes directly in Odoo Point of Sale. This module supports multiple payment QR formats — ABA KHQR, WeChat Pay, and Alipay — by displaying the payment QR on the POS customer screen or printed bill (printed bill QR is only available for the POS Restaurant module).

Features

  • Dynamic QR code generation at checkout using ABA PayWay API
  • Supports ABA KHQR, WeChat Pay, and Alipay formats
  • Display QR on the POS cashier screen and customer-facing display
  • Print QR code on restaurant bills (POS Restaurant module required)
  • Configurable per payment method and per Point of Sale

Getting Started

  1. Go to Contacts → Configuration → Bank Accounts, select the bank account you want to accept payments with, then fill in your API Key and Merchant ID as provided by ABA PayWay.

Configuration

  1. Go to POS module → Configuration → Payment Methods.
  2. Click Add New, then in the Integration section:
    • Choose Bank App (QR Code)
    • Select a QR format: ABA KHQR, WeChat Pay, or Alipay
    • Make sure to use the same name for the Payment Method Name
    • Configure the Bank Journal as usual
  3. To print the payment QR on bills, enable "Allow QR on bills" (works with Restaurant POS only), then select the Point of Sale to apply it to.
